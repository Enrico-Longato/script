# Company Data Processing Scripts

Data cleaning and preparation scripts for regional company.

## Files

- **01_anagrafica.ipynb** - Interactive Jupyter notebook for data processing - company registry data (IMPRESE.xlsx)
- **01_anagrafica.py** - Python script version of the notebook - company registry data (IMPRESE.xlsx)

## Description

These scripts import data from Excel files and produce clean, processed data in CSV format for multiple outputs:
- 01_anagrafica.* - Cleaned company registry data (IMPRESE.csv)    
    - Repository version
    - Innovation Intelligence versions (with and without FVG regional filtering)
    - Non-capital companies versions
    - Fabio Morea, Leyla Vesnic, Enrico Longato @ Area Science Park
    - Original repository: https://github.com/areasciencepark/I2FVG_Anagrafiche
